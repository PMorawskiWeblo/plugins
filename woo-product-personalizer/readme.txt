Personalize WooCommerce products with layout-driven image and text editing, live preview, and order production files.

== Description ==

Woo Product Personalizer adds per-product personalization using admin-configured layouts. Customers can upload images, enter text, transform photos on a canvas preview, and add personalized products to cart. Orders receive generated project files.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/woo-product-personalizer/`
2. Activate through the Plugins screen
3. Ensure WooCommerce is active
4. Configure settings under WooCommerce → Product Personalizer
5. Create layouts and assign them to products
